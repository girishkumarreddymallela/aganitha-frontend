# URL Shortening APP Frontend
# Built with vite develop,emt tool with react and shadcn components
# steps to run
  1. configure .env.local  with variable name as mentioned in .env.example (we need to add backend url where it is running)
  2. This project is using pnpm as package manager so need to install pnpm if not present (npm i -g pnpm)
  3. next step is to install packages (pnpm i)
  4. and Run the project (pnpm dev).
